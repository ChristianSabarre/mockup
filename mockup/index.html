System Flow /div>
APE PIPELINE
  1. Admin (Setup)
    Define stations and required fields per station.
  2. Registration (Onboard)
    Register patient → verify ID → issue QR → start APE session → print voucher/consent.
  3. Nurse (Vitals)
    Scan QR → capture vitals → verify station transaction.
  4. Lab (Diagnostics)
    Scan QR → create order → assign container → encode results → release.
  5. Doctor (Assessment)
    Review record → exam findings → physician signature → complete + seal.
  6. Patient Portal (View)
    Login → view QR + session summaries → sign consent if needed.
